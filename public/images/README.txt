===================================================
PETUNJUK MENGGANTI FOTO POLAROID BESTIE DI ALBUM REMINDER
===================================================

Halo Bestie! Kamu bisa mengganti foto-foto default di album dengan foto asli kamu dan sahabatmu sangat mudah:

1. Siapkan 3 foto terbaik kalian berdua.
2. Ubah format/ekstensi foto menjadi JPG (.jpg).
3. Ubah nama file foto tersebut menjadi:
   - polaroid_1.jpg (untuk halaman polaroid pertama)
   - polaroid_2.jpg (untuk halaman polaroid kedua)
   - polaroid_3.jpg (untuk halaman polaroid ketiga)
4. Buang atau overwrite file-file yang ada di dalam folder `/public/images/` ini dengan file foto yang baru kamu buat dengan nama yang sama persis.
5. Selesai! Web akan mendeteksi dan menampilkan foto indah kalian berdua secara otomatis tanpa perlu mengubah baris kode apapun! 🎉✨
